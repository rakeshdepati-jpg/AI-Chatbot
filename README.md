# 🤖 AI Chatbot

An AI-powered chatbot built using Flask, HTML, CSS, JavaScript, and OpenRouter API. The chatbot can answer user questions, display a typing animation, and save chat history.

## 🚀 Features

- AI-powered responses
- Chat history saved in JSON
- New Chat button
- Delete Chat button
- Typing animation
- Modern responsive UI
- Custom background

## 🛠️ Technologies Used

- Python
- Flask
- HTML
- CSS
- JavaScript
- OpenRouter API

## 📁 Project Structure

```
AI-Chatbot/

```

## ▶️ Installation

1. Clone the repository
2. Install dependencies

```bash
pip install -r requirements.txt
```

3. Add your OpenRouter API key in `config.py`

```python
API_KEY = "YOUR_API_KEY"
```

4. Run the project

```bash
python app.py
```

5. Open your browser and visit:

```
http://127.0.0.1:5000
```
